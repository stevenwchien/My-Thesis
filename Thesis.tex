% The Abstract Page
\addtotoc{Abstract}  % Add the "Abstract" page entry to the Contents
\abstract{
\addtocontents{toc}{\vspace{1em}}  % Add a gap in the Contents, for aesthetics

Many of the restrictions on quantum secret sharing schemes are derived from the no-cloning theorem. In this paper, we explore the potential benefit of implementing quantum secret sharing schemes using two or more identical copies of a secret quantum state. The hope is that this can be used to circumvent the no-cloning theorem. Our approach involves taking the union of two or more access structures, one for each quantum state, in order to implement an access structure that would otherwise not be realizable. Using this method, we find that we are indeed able to implement more access structures than we did before, but only one new threshold scheme for every new copy of the share, given a fixed size of the pool of players.

}
